Overview

Pytorch code to reproduce the results of the clustring algorithms IMSAT and Deep_RIM proposed in [1]. The adoped data set is MNIST. The implementation in [1] is based on Chainer.

Dependencies

Package version used:
python 3.6.6
torch 0.4.1

Train model:
To run Deep_RIM 
$ python Deep_RIM.py

To run IMSAT 
$ python Imsat.py
